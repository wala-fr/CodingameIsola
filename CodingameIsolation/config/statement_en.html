  The Goal

The goal of the game is to block the opponent pawn.

  Rules

Board:
  • The game is played on a 7x7 board.
  • Player 0 always starts at (0, 3) and player 1 at (6, 3).
At each turn:
  1. You must move your pawn to an adjacent tile (diagonally included).
    You can't :
    • stay put
    • move to the tile occupied by the opponent pawn
    • move to an already removed tile.

  2. You must remove an accessible tile.
    You can't remove a tile occupied by a pawn.
Victory Conditions
  • The opponent pawn can't move.
Loss Conditions
  • Your pawn can't move.
  • You do not respond in time or output an invalid action.

  Game Input

Initialization input
First line: id : The id of the player.
Second line: playerPositionX : the x coordinate of your pawn.
Second line: playerPositionY : the y coordinate of your pawn.
Input for one game turn
Next opponentPositionX: the x coordinate of the opponent pawn.
Next opponentPositionY: the y coordinate of the opponent pawn.
Next opponentLastRemovedTileX: the x coordinate of the last tile removed by the opponent (-1 if no tile has been removed (first round)).
Next opponentLastRemovedTileY: the y coordinate of the last tile removed by the opponent (-1 if no tile has been removed (first round)).
Output
A single line containing the coordinates where you want to move your pawn, followed by the coordinates of the tile you want to remove. Example "3 1 3 5".
You can also print messages by doing the following. Example: "3 1 3 5;message".
Constraints
Response time first turn is ≤ 1000 ms.
Response time per turn is ≤ 100 ms.
